# Apk publisher
CLI tool for publish and management application in Google Play Console.

Based on [Google Play Java library](https://developers.google.com/android-publisher/api-ref)

## Commands
- `apks`
  - `list`
  - `upload`
- `bundles`
  - `list`
  - `upload`
- `deobfuscationfiles`
  - `upload`
- `tracks`
  - `list`
  - `get`
  - `patch`
  - `update`
